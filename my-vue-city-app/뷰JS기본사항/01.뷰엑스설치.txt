[ 01. 뷰엑스 설치하기 ]

1.기본 webpack에는 Vuex 라이브러리가 설치되어 있지 않음
2.Vue CLI로 설치 명령어를 실행하여 설치하자
((기본 npm 설치 명령어))
-npm install 설치할패키지이름@버전
ex) npm install --save --force vuex@3 babel-polyfill
-vuex와 babel을 같이 설치하고 싶은 경우, 한꺼번에 쓰면 됨
-혹시나 잘못 설치한 경우, 다시 설치하면 덮어쓰기가 됨

1) npm install
-기본 설치 명령어
2) 옵션1 : --save (필수)
-설치 레코드 저장
-package.json 파일의 "dependencies"에 기록되고 관리되도록 함
-현재 최신 모듈 npm은 --save를 안 써도 자동으로 등록해주지만 우리는 쓰도록 하기
3) 옵션2 : --force (필수아님)
-강제 설치
-기존에 같은 종류가 설치된 것을 덮어 쓸 때, 그밖에 설치가 자꾸 안 될 때 쓰면 강제 설치가 됨
4) vuex@3
-vuex 설치시 버전 3.x로 설치함
-만약 버전을 안 쓰면 최신 버전을 설치하므로, 버전을 맞춰서 설치해야하는 경우 @x를 구체적으로 써줘야 함
5) babel-polyfill
-babel은 브라우저 호환성때문에 설치함
-좀 옛날 브라우저도 최신 코드가 실행되도록 변환해주는 것임
ex) ES5, ES6 JS문법을 이해 못하는 브라우저에 적용되도록 문법적 변환해주기 때문에 예전 JS문법을 쓸 필요가 없음
(안 깔아도 되지만, 혹시나 생길 문제 예방 위함)
-뷰엑스와 함께 설치됨
